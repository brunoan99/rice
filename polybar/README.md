# **Polybar**

## **Installation**

Install polybar

```sh
位 sudo pacman -S polybar
```

## **Setup**

Frist create the config folder

```sh
位 mkdir -p $HOME/.config/polybar
```

Link config files to the config folder

```sh
位 ln -sf $HOME/.rice/polybar/config.ini $HOME/.config/polybar/config.ini
```

```sh
位 ln -sf $HOME/.rice/polybar/launch.sh $HOME/.config/polybar/launch.sh
```

## **Configuration**

To change any spec of configuration uses the file in this repo instead of change in config folder directly
